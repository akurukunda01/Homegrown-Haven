CURRENT SETUP: Groq — Llama 4 Scout
-------------------------------------
  Model:      meta-llama/llama-4-scout-17b-16e-instruct
  Provider:   Groq (LPU hardware)
  API Key:    GROQ_API_KEY (set in .env)
  Speed:      ~0.2-0.3s TTFT
  Free Tier:  30,000 tokens/min, 500,000 tokens/day

Note: HuggingFace free tier was tried first but has a monthly credit cap
that depleted quickly under voice agent usage (402 error).


HOW TO ROLL BACK TO LM STUDIO
------------------------------
In voice_chat.py, find the LLM CONFIGURATION block and swap the comments:

  1. Comment out the HuggingFace llm=openai.LLM(...) block
  2. Uncomment the ROLLBACK block below it

LM Studio must be running locally at http://localhost:1234 with a model loaded.


OTHER AVAILABLE OPTIONS (for future switching)
----------------------------------------------
All options use the openai.LLM plugin — only model/api_key/base_url change.

Option A — Groq (fastest, already imported):
  llm=groq.LLM(
      model="llama-3.3-70b-versatile",
      api_key=GROQ_API_KEY,
  )
  Speed: ~0.2-0.3s TTFT. Requires GROQ_API_KEY in .env (already set).

Option B — OpenAI gpt-4o-mini:
  llm=openai.LLM(
      model="gpt-4o-mini",
      api_key=OPENAI_API_KEY,
  )
  Speed: ~0.3-0.5s TTFT. Requires OPENAI_API_KEY in .env (already set).

Option C — HuggingFace (current), smaller/faster model:
  Change model to: "meta-llama/Llama-3.1-8B-Instruct"
  Same base_url and HF_TOKEN. Faster but less capable.
